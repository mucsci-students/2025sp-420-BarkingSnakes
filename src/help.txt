Options
help
load <filename>
save <filename>
quit
exit
class <classname> (brings up submenu of options for class)
    add
    delete (brings up Y/N prompt)
    attribute add <attributename>
    attribute delete <attributename>
    attribute rename <oldattributename> <newattributename>
relation add <sourceclass> <destclass>
relation delete <sourceclass> <destclass>
-classname will be invalid if any of these keywords, or python 
    language keywords, and no action will be taken
-input will also be invalid if classname specified does not
    exist for delete, rename, or relation actions
